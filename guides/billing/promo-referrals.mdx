A Timely.ai tem dois mecanismos para reduzir custos ou ganhar créditos extras: **códigos promocionais** aplicados no checkout e o **programa de indicação**, que recompensa você a cada novo usuário que se cadastra pelo seu link.

## Códigos promocionais

Códigos promocionais oferecem desconto percentual ou valor fixo em BRL sobre o plano assinado. Eles são aplicados diretamente no Stripe durante o checkout.

### Como aplicar um código

<Steps>
  <Step title="Inicie o checkout">
    Na página de Billing, escolha um plano e clique em **Assinar** ou **Upgrade**.
  </Step>
  <Step title="Insira o código na tela de checkout">
    Na sessão de checkout do Stripe, há um campo para **Código promocional**. Cole o código e clique em **Aplicar**.
  </Step>
  <Step title="Confirme o desconto">
    O valor com desconto é exibido antes de confirmar o pagamento. Se o código for válido, o desconto aparece imediatamente.
  </Step>
</Steps>

### Regras de uso

- Cada código só pode ser usado **uma vez por empresa**
- Códigos têm data de validade — após expirar, não são aceitos
- Alguns códigos têm limite total de usos (ex: primeiros 100 clientes)
- Certos códigos são válidos apenas para planos ou ciclos específicos (mensal ou anual)

### Mensagens de erro comuns

| Mensagem | Causa |
|---|---|
| Código não encontrado ou inativo | O código não existe ou foi desativado |
| Código expirado | A data de validade passou |
| Código atingiu o limite de uso | O número máximo de usos foi atingido |
| Você já utilizou este código | A empresa já resgatou esse código anteriormente |
| Código não válido para o plano selecionado | O código é restrito a outro plano |
| Código válido apenas para assinaturas anuais | O ciclo selecionado não é compatível |

## Programa de indicação

O programa de indicação recompensa você com créditos (Coins) a cada novo cadastro feito pelo seu link exclusivo. Quanto mais indicações, maior o nível de recompensa.

### Como funciona

<Steps>
  <Step title="Acesse seu link de indicação">
    No menu lateral, clique em **Indique e Ganhe** ou acesse **Configurações → Programa de Indicação**.
  </Step>
  <Step title="Compartilhe o link">
    Copie o link ou compartilhe diretamente pelo WhatsApp. O link é único e permanente para sua conta.
  </Step>
  <Step title="Acompanhe as indicações">
    A cada novo usuário que criar conta usando seu link, você ganha créditos automaticamente. O painel mostra o total de indicações creditadas e seu nível atual.
  </Step>
</Steps>

### Recompensas por indicação

Você ganha créditos a cada cadastro confirmado pelo seu link, até o limite máximo de indicações creditadas.

### Níveis de recompensa

O programa tem quatro níveis baseados no total acumulado de indicações:

| Nível | Indicações necessárias |
|---|---|
| Bronze | A partir de 1 indicação |
| Prata | A partir de 5 indicações |
| Ouro | A partir de 10 indicações |
| Platina | A partir de 15 indicações |

Os níveis são exibidos no painel de indicações com o progresso em tempo real. Ao atingir o nível Platina, o limite de créditos por indicações é atingido — mas seu link continua funcionando para novos cadastros.

<Note>
  Créditos de indicação são adicionados ao pool de créditos extras da sua conta — sem data de validade e sem vínculo com o ciclo de cobrança do plano.
</Note>

### Compartilhando pelo WhatsApp

O painel de indicações tem um botão de compartilhamento direto para o WhatsApp com uma mensagem pré-formatada:

> "Estou usando o Timely.ai para automatizar meu atendimento com IA! Cria sua conta gratuita pelo meu link e ganha créditos bônus: [link]"

Você pode editar a mensagem antes de enviar caso queira personalizar o convite.

### Perguntas frequentes

**Quando os créditos são creditados?**
Logo após o novo usuário concluir o cadastro e ativar a conta pelo seu link de indicação.

**Meu link pode ser compartilhado quantas vezes?**
Ilimitadas. Mas os créditos de recompensa têm um teto máximo de indicações creditadas por conta.

**Indicações que não criaram conta contam?**
Não. Apenas cadastros concluídos e validados geram créditos para você.

**Posso usar um código promocional e também um link de indicação?**
Sim. São mecanismos independentes. O código promocional dá desconto no plano; o link de indicação gera créditos para quem indicou.
