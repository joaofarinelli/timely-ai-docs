---
title: "Configurações do agente"
description: "Resumo de todas as seções de configuração disponíveis para um agente na Timely.ai e para que cada uma serve."
---

Cada agente possui um conjunto de configurações que você acessa clicando no agente na lista e navegando pelas abas disponíveis. Alterações entram em vigor imediatamente para novas conversas — conversas já abertas não são afetadas.

## Resumo

| Seção | O que configura |
|-------|----------------|
| [Modelo](/guides/agents/settings/model) | Provedor de LLM (OpenAI, Anthropic, Google, OpenRouter), modelo específico, temperatura e effort |
| [Nome interno](/guides/agents/settings/internal-name) | Identificador do agente dentro da plataforma — visível para a equipe, não para o cliente |
| [Role](/guides/agents/settings/role) | Função e persona do agente — o "cargo" que ele ocupa na operação |
| [Prompt](/guides/agents/settings/prompt) | Instruções de sistema com editor de IA, histórico de versões e análise de qualidade em tempo real |
| [Temperatura](/guides/agents/settings/temperature) | Grau de criatividade vs consistência das respostas (0,00 a 1,00) |
| [Effort](/guides/agents/settings/effort) | Profundidade de raciocínio controlada por número máximo de passos (10 a 100) |
| [Capacidades](/guides/agents/settings/capacities) | Funcionalidades opcionais: planejamento, raciocínio, data e hora automática |
| [Prompts condicionais](/guides/agents/settings/conditional-prompts) | Blocos de instrução ativados por variáveis da conversa ou do contato |
| [Chat interno](/guides/agents/settings/internal-chat) | Interface de teste direto com o agente sem passar por canal externo |
