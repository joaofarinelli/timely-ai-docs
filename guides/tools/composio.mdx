Tools são integrações com apps externos que permitem ao agente executar ações reais além de responder texto. Com uma tool conectada, o agente pode criar eventos no Google Calendar, registrar leads no HubSpot, enviar mensagens no Slack ou consultar pedidos no Shopify — tudo dentro de uma conversa.

## Visão geral

Todas as integrações são fornecidas via **Composio**, uma plataforma de conectores que abstrai a autenticação e a execução das ferramentas. Na Timely.ai, você gerencia as tools na aba **Tools** do agente.

## Como conectar um app

<Steps>
  <Step title="Abra o catálogo de tools">
    Na aba **Tools** do agente, clique em **Adicionar tool** para abrir o catálogo de apps disponíveis.
  </Step>
  <Step title="Selecione o app">
    Navegue pelo catálogo ou use a busca por nome. Clique no card do app desejado.
  </Step>
  <Step title="Autentique o app">
    Dependendo do app, a autenticação é feita de formas diferentes:

    <AccordionGroup>
      <Accordion title="OAuth (maioria dos apps)">
        Clique em **Conectar via OAuth**. Uma janela do navegador abre para você autorizar o acesso na conta do app. Após autorizar, a conexão é estabelecida automaticamente.

        Apps com OAuth: Notion, Google Calendar, Google Meet, Google Sheets, ClickUp, Trello, Asana, Airtable, Canva, GitHub, Jira, Linear, Supabase, Shopify, HubSpot, Salesforce, Zendesk, Slack, Gmail, LinkedIn.
      </Accordion>
      <Accordion title="Credenciais próprias (apps selecionados)">
        Alguns apps exigem que você crie um app de desenvolvedor próprio e informe as credenciais. Nesses casos, um formulário é exibido com os campos necessários.

        Apps com credenciais próprias: **Twitter/X** (Client ID e Client Secret OAuth 2.0), **Meta Ads** (App ID e App Secret), **Apollo.io** (API Key), **Kommo** (Integration ID e Secret Key), **Browse.ai** (API Key), **Apify** (OAuth DCR).
      </Accordion>
      <Accordion title="Sem autenticação">
        **Browser Tool** não requer autenticação — está disponível imediatamente após ser adicionado ao agente.
      </Accordion>
    </AccordionGroup>
  </Step>
  <Step title="Selecione as tools">
    Após conectar, escolha quais ferramentas específicas do app o agente pode usar. Habilitar apenas o necessário evita que o agente chame tools inadequadas.
  </Step>
  <Step title="Salve e teste">
    Clique em **Salvar**. Use o [Playground](/guides/agents/settings/internal-chat) para testar se o agente aciona as tools corretamente.
  </Step>
</Steps>

## Configuração por tool

Dentro de cada app conectado, você pode configurar cada tool individualmente pelo **Catálogo de Tools**:

- **Ativar/desativar**: controla se o agente pode usar aquela tool específica.
- **Descrição customizada**: reescreva a descrição da tool para dar contexto adicional ao modelo sobre quando usá-la.
- **Parâmetros padrão**: alguns parâmetros podem ser fixados com valores padrão (ex.: sempre usar o mesmo calendário).

## Tool catalog

O **Tool Catalog** é uma visão consolidada de todas as tools habilitadas no agente — de todos os apps conectados. Use para:

- Ver quais tools estão ativas de forma centralizada
- Reordenar a prioridade de tools quando apps têm ferramentas com funções similares
- Identificar tools que podem causar conflito de comportamento

## Boas práticas

<Note>
Habilite apenas as tools que o agente realmente precisa. Cada tool disponível adiciona contexto ao prompt de raciocínio do modelo — muitas tools desnecessárias podem confundir o agente e aumentar o custo por execução.
</Note>

- Para apps de CRM (HubSpot, Kommo, Salesforce), comece com tools de leitura (list, get, search) antes de habilitar escrita (create, update, delete).
- Use a descrição customizada de cada tool para guiar o agente: "Use esta tool apenas quando o cliente perguntar sobre o status de um pedido específico".
- Teste cada tool individualmente no Playground antes de combinar múltiplos apps.
- Apps que requerem credenciais próprias (Twitter/X, Meta Ads, Apollo.io, Kommo) exigem configuração no portal do desenvolvedor do respectivo serviço antes de conectar.
