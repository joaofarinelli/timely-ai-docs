Workflows são sequências de nós conectados que executam automações na Timely.ai. Cada fluxo começa em um trigger, percorre nós de ação e lógica configurados por você e pode retornar dados estruturados ao chamador quando necessário.

## Como os Workflows Funcionam

Um workflow é montado no editor visual drag-and-drop: arraste componentes para o canvas, conecte os nós pelos handles de saída e configure cada um com um duplo clique. Toda publicação gera uma versão imutável — o rascunho em edição não afeta o fluxo em produção até você clicar em **Publicar**.

Cada execução gera um **Workflow Run** com status, log individual de cada nó, payload de entrada e saída e duração total. O histórico fica disponível em **Automações > Workflows > Ver execuções**.

## Nós Disponíveis

O editor oferece três categorias de nós:

- **Webhook** — recebe requisição HTTP externa e expõe body, headers e query params para os nós seguintes
- **Agendamento** — dispara em horário programado via expressão cron com fuso horário configurável
- **Requisição do Agente** — permite que um agente IA acione o workflow como ferramenta, passando parâmetros estruturados
- **Requisição HTTP** — chama qualquer API REST externa com suporte a GET, POST, PUT, PATCH e DELETE
- **Definir Variável** — cria ou transforma variáveis acessíveis via `{{ $vars.nome }}`
- **Código JavaScript** — executa lógica customizada em ambiente isolado com timeout de 10 segundos
- **Transformação IA** — processa texto com modelo de linguagem para extrair, classificar ou reescrever dados

## Por Que Usar Workflows

Workflows são a escolha certa quando você precisa integrar sistemas externos, processar dados antes de agir ou orquestrar passos que dependem uns dos outros. Diferente de instruções no prompt do agente, um workflow executa de forma determinística, rastreável e auditável — cada nó tem logs, status e tempo de execução registrados.

Use workflows para automatizar notificações baseadas em eventos, sincronizar dados entre plataformas ou gerar relatórios periódicos sem intervenção manual.
