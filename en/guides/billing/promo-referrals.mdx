Timely.ai has two mechanisms to reduce costs or earn extra Credits: **promo codes** applied at checkout and the **referral program**, which rewards you each time a new user signs up through your unique link.

## Promo codes

Promo codes offer a percentage discount or a fixed amount off the subscribed plan. They are applied directly in Stripe during checkout.

### How to apply a code

<Steps>
  <Step title="Start checkout">
    On the Billing page, choose a plan and click **Subscribe** or **Upgrade**.
  </Step>
  <Step title="Enter the code at the checkout screen">
    In the Stripe checkout session, there is a field for **Promotional code**. Paste the code and click **Apply**.
  </Step>
  <Step title="Confirm the discount">
    The discounted amount is displayed before confirming payment. If the code is valid, the discount appears immediately.
  </Step>
</Steps>

### Usage rules

- Each code can only be used **once per company**
- Codes have an expiration date — after expiry, they are not accepted
- Some codes have a total usage limit (e.g., first 100 customers)
- Certain codes are valid only for specific plans or billing cycles (monthly or annual)

### Common error messages

| Message | Cause |
|---|---|
| Code not found or inactive | The code does not exist or has been deactivated |
| Code expired | The expiration date has passed |
| Code has reached its usage limit | The maximum number of uses has been reached |
| You have already used this code | The company has already redeemed this code |
| Code not valid for the selected plan | The code is restricted to another plan |
| Code valid for annual subscriptions only | The selected billing cycle is not compatible |

## Referral program

The referral program rewards you with Credits (Coins) for each new signup made through your unique link. The more referrals, the higher your reward level.

### How it works

<Steps>
  <Step title="Access your referral link">
    In the sidebar, click **Refer & Earn** or go to **Settings → Referral Program**.
  </Step>
  <Step title="Share the link">
    Copy the link or share it directly via WhatsApp. The link is unique and permanent for your account.
  </Step>
  <Step title="Track your referrals">
    Each time a new user creates an account using your link, you automatically earn Credits. The panel shows the total credited referrals and your current level.
  </Step>
</Steps>

### Rewards per referral

You earn Credits for each confirmed signup through your link, up to the maximum number of credited referrals.

### Reward levels

The program has four levels based on the accumulated total of referrals:

| Level | Referrals required |
|---|---|
| Bronze | Starting from 1 referral |
| Silver | Starting from 5 referrals |
| Gold | Starting from 10 referrals |
| Platinum | Starting from 15 referrals |

Levels are displayed in the referrals panel with real-time progress. Upon reaching the Platinum level, the Credits cap for referrals is reached — but your link continues to work for new signups.

<Note>
  Referral Credits are added to your account's extra Credits pool — with no expiration date and no link to the plan billing cycle.
</Note>

### Sharing via WhatsApp

The referrals panel has a direct sharing button for WhatsApp with a pre-formatted message:

> "I'm using Timely.ai to automate my customer service with AI! Create your free account through my link and get bonus Credits: [link]"

You can edit the message before sending if you want to personalize the invite.

### Frequently asked questions

**When are Credits credited?**
Shortly after the new user completes signup and activates their account through your referral link.

**How many times can my link be shared?**
Unlimited. But the reward Credits have a maximum cap of credited referrals per account.

**Do referrals that did not create an account count?**
No. Only completed and validated signups generate Credits for you.

**Can I use a promo code and also a referral link?**
Yes. They are independent mechanisms. The promo code gives a discount on the plan; the referral link generates Credits for the referrer.
