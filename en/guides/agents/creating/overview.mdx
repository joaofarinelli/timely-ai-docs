---
title: "Creating an Agent"
description: "Two paths to create an agent in Timely.ai: configure each field manually or let the conversational assistant guide the process."
---

Timely.ai offers two agent creation flows. Choose the one that best fits your situation:

- [Manual creation (from scratch)](/en/guides/agents/creating/from-scratch) — fill in the fields directly with full control from the first step. Ideal for those who already know exactly what the agent should do.
- [Creation with the conversational assistant](/en/guides/agents/creating/wizard) — answer questions in natural language and the assistant configures and generates the prompt automatically. Ideal for your first agent or for use cases that are still being structured.
