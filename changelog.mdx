Todas as mudanças relevantes ficam registradas aqui em ordem cronológica reversa. Breaking changes da API são destacadas com <span style={{ color: "#dc2626", fontWeight: 600 }}>⚠ Breaking</span>.

## 2026-04-19

### Documentação
- **Nova estrutura de documentação** inspirada em padrões modernos (Mintlify): tabs separados para Comece Aqui, Guias de Produto, API Reference, Webhooks, AI e HTTP Collections.
- **Catálogo de eventos de webhook** completo com payloads de exemplo por tipo.
- **Glossário técnico** com os principais termos da plataforma.
- **llms.txt** disponível em `/llms.txt` para uso em IDEs com IA (Cursor, Claude, VS Code).

## 2026-04-18

### Workers
- Upload de arquivos e imagens no chat do Worker com suporte multimodal.
- Melhorias de UX no fluxo de mensagens e testes automatizados.

### Settings
- Aba de API movida para página dedicada na sidebar de configurações.
- Correção no carregamento do perfil de usuário após troca de workspace.
